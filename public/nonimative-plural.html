Slovak Noun Pluralization & Stem Softening

This module provides tools to derive stems from Slovak nouns and generate their nominative plural forms, including softening rules for consonants at the stem-suffix boundary.

Stemmer

deriveStem(word, gender) extracts the base stem from a Slovak noun.

Rules applied:

  • Remove singular endings like -a, -o, -e, -ie, -um
  • Returns the raw stem without softening
deriveStem("vysvedčenie", "N"); // → "vysvedčen"
deriveStem("hrdina", "M");      // → "hrdin"
deriveStem("kosť", "F");        // → "kosť"

Softening is not applied in the stemmer. It is handled separately in the plural generator.

Nominative Plural Generator

nominativePlural(word, gender) returns the nominative plural of a Slovak noun.

Rules by Gender

Masculine

  • Consonant-ending: append i with softening
  • -a ending: append ovia, no softening

Feminine

  • -a ending: append y
  • -ia ending: append i
  • Consonant-ending: append i with softening

Neuter

  • -o: append á
  • -e: append ia
  • -ie: append ia
  • -um: append á
  • Consonant-ending: usually unchanged, softening skipped
---

Examples

nominativePlural("chlap", "M");        // → "chlapi"
nominativePlural("princ", "M");        // → "princovia"
nominativePlural("hrdina", "M");       // → "hrdinovia"
nominativePlural("kolega", "M");       // → "kolegovia"

nominativePlural("žena", "F");         // → "ženy"
nominativePlural("chémia", "F");       // → "chémii"
nominativePlural("kosť", "F");         // → "kosti"

nominativePlural("mesto", "N");        // → "mestá"
nominativePlural("srdce", "N");        // → "srdcia"
nominativePlural("vysvedčenie", "N");  // → "vysvedčenia"
nominativePlural("minimum", "N");      // → "minimá"
---

Edge Cases

  • Single-letter nouns (e.g., "k" → "ki")
  • Feminine consonant-ending nouns: softening handled
  • Masculine nouns ending in -a: softening skipped
  • Irregulars like dieťa → deti or človek → ľudia should be handled via a lookup table
---

Usage

import { nominativePlural } from "./NominativeDerivations.js";

const plural = nominativePlural("kosť", "F");
console.log(plural); // → "kosti"

This module can be used to generate plurals or as a foundation for building **full declension tables** for Slovak nouns.
