# Nominative Case
Base dictionary form.  
Used for subjects and predicate nouns.

---

## 1. Noun Stems

### Masculine
- Consonant-ending kept: *hrad → hrad*
- -a → remove **a**: *kolega → koleg-*
- -o/-e remove vowel  
- -ec animate → drop **e**: *chlapec → chlapc-*

### Feminine
- -a → remove **a**
- -ia → remove **ia**
- consonant-ending unchanged

### Neuter
- -o/-e → remove vowel
- -ie → remove **ie** → **i**
- -um → remove **um**

</section>


<section id="nouns-singular">

## 2. Nouns — Nominative Singular

### Masculine
- base form: *chlap, muž, hrdina, stroj*

### Feminine
- base form: *žena, kniha, kosť*

### Neuter
- base form: *mesto, auto, srdce*

</section>


<section id="nouns-plural">

## 3. Nouns — Nominative Plural

### Masculine
- animate → **-i**: *chlapi, muži*
- masc ending -a → **-ovia**: *hrdinovia, kolegovia*
- inanimate hard → **-y**: *hrady, plány*
- inanimate soft → **-e**: *stroje*

### Feminine
- -a → **-y**
- -ia → **-ie**
- consonant-ending → **-i**

### Neuter
- -o → **-á**
- -e → **-ia**
- -ie → **-ia**
- -um → **-á**

</section>


<section id="adjectives">

## 4. Adjectives — Nominative

### Singular
- M: *dobrý chlap*
- F: *dobrá žena*
- N: *dobré mesto*

### Plural
- M animate: *dobrí muži*
- M inanimate: *dobré hrady*
- F: *dobré ženy*
- N: *dobré mestá*

</section>


<section id="demonstratives">

## 5. Demonstratives — Nominative

### Singular
- M: *ten*
- F: *tá*
- N: *to*

### Plural
- Masc animate: *tí*
- Masc inanimate: *tie*
- F + N: *tie*

</section>
