Slovak cases — compact student guide

For each case below: Noun (singular → plural) then Adjective (singular → plural). Check specific endings first, then apply general fallbacks.

Accusative

Noun — singular

  • Masculine animate: if nominative ends -a → replace with -u; otherwise add -a.
  • Masculine inanimate: same as nominative (no change).
  • Feminine: -a → -u; -ia → -iu; consonant‑ending feminine stays nominative.
  • Neuter: same as nominative.

Noun — plural

  • Masculine animate: accusative plural = stem + -ov (verify stem formation separately).
  • Masculine inanimate: accusative plural = nominative plural.
  • Feminine & Neuter: accusative plural = nominative plural.

Adjective — singular

  • Classify adjective: HARD (ends -ý, -á, -é) or SOFT (ends -í or forms with -ia/-ie); remove final vowel to get stem.
  • Masculine: HARD animate → -ého; HARD inanimate → -ý; SOFT animate → -ieho; SOFT inanimate → -ý.
  • Feminine: HARD → -ú; SOFT → -iu.
  • Neuter: → -é.

Adjective — plural

  • Masculine: HARD animate → -ých; HARD inanimate → -é; SOFT animate → -ich; SOFT inanimate → -ie.
  • Feminine: SOFT → -ie; otherwise → -é.
  • Neuter: SOFT → -ie; otherwise → -é.

Instrumental

Noun — singular

  • Masculine: -ec → -com; otherwise add -om.
  • Feminine: -a → -ou; -ia → -iou; otherwise add -ou.
  • Neuter: -ie → -ím; -o → -om; otherwise add -om.

Noun — plural

  • All genders: instrumental plural → add -mi to the plural form/stem.

Adjective — singular

  • Masculine & Neuter: HARD → -ým; SOFT → -ím.
  • Feminine: → -ou.

Adjective — plural

  • All genders: HARD → -ými; SOFT → -imi.

Locative

Noun — singular

  • Masculine animate: add -ovi.
  • Masculine inanimate: depends on final consonant: k/g/h/ch → -u; d/l/r → -e; otherwise -i.
  • Feminine: -a → -e; -ia → -ii; consonant‑ending feminine add -i.
  • Neuter: -o → -e; -ie → -í; otherwise add -e.

Noun — plural

  • Masculine: hard stems → -och.
  • Feminine: -a → -ách; -ia → -iách; consonant → -ach.
  • Neuter: -o → -ách; -ie → -iach; -e → -ách.

Adjective — singular

  • Masculine & Neuter: → -om.
  • Feminine: → -ej.

Adjective — plural

  • All genders: HARD → -ých; SOFT → -ich.

Example: médium / média

Neuter noun from Latin; singular médium, plural média.

Selected forms
  • Nominative singular: médium
  • Nominative plural: média
  • Locative singular: médiu → use: o médiu (about the medium)
  • Locative plural: médiách → use: o médiách (about the media)
  • Instrumental singular: médiom
  • Instrumental plural: médiami

General checklist

  • Test specific endings first (-ec, -ia, -ie, -a, -o), then apply general suffix rules.
  • Decide masculine animacy — it affects accusative and some plural patterns.
  • Adjective workflow: classify HARD vs SOFT, remove final vowel for stem, then add the matching ending.
  • Learn nominative plural formation — many plural case forms depend on it.

This page covers common regular patterns. Irregular nouns and exceptions exist; consult a full declension table for specific nouns when needed.
